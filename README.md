# 学生作业管理系统
	
**实现功能：**
	
	学生上传文件到系统中，后台统一命名存储文件，管理员登陆后可以批量下载后台打包过后的文件。
	
**项目所用框架：**

前端：
	
	1.JQuery
	2.Bootstrap
	3.Bootstrap-fileinput
后端：
	
	1.SpringMVC
	2.Spring
	3.MyBatis
	4.Shiro
**不足：**

	1.业务代码几乎全写在控制层
	2.全部下载时浏览器获取不到文件大小
